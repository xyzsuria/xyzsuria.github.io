抽象基类(abc模板)

抽象基类：在一个基础的类中设定一些方法，所有继承这个基类的类都必须覆盖这个基类中的方法；无法用来实例化。

应用场景
  1. 检查某个类是否使用某个方法
  2. 我们需要强制某个子类必须实现某些方法

    列表/生成器/字典推导式

    列表生成式（列表推导式）
    列表推导式是通过一行代码生成列表。
    列表生成式性能高于列表操作
    odd_list = [i for in range(20) if i%2==1]
    生成器表达式
    odd_gen = (i for i in range(20) if i % 2 == 1 )
    odd_gen的类型为generator。
    生成器可以转换为列表类型，odd_list = list(odd_gen)。
